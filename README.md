### VisualizeArt
<p>react를 이용한 그림 색상 분석 웹 사이트</p>

<h2>개요</h2>
<span>자신의 그림을 업로드하여 그 그림에 색감(대표색 외 팔레트색)에 대한 시각적인 그래프와 색에 대한 음성기능을 통해 색에 대한 정보를 알 수 있는 프로젝트</span>
<br>
<h3>개발 기간: 24.2 ~ 24.3<h3>
<br>

개발 언어
<li>react</li>
<li>TypeScript</li>
<li>scss</li>
<br>

개발 도구
